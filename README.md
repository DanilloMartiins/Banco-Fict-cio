# Banco Fictício AAA

## Sobre
Projeto desenvolvido para ensinar conceitos de programação em cursos. Este projeto simula operações bancárias básicas, como visualizar saldo, transferir valores e receber transferências. Ideal para praticar conceitos de programação em Java, como estruturas condicionais, loops e manipulação de entrada de usuário.

## Funcionalidades

- **Visualização de Saldo:** Permite que o usuário veja o saldo disponível na conta.
- **Transferência de Valores:** Permite que o usuário transfira valores de sua conta, com verificação de saldo disponível.
- **Receber Transferência:** Permite que o usuário receba valores em sua conta, atualizando o saldo disponível.
- **Menu Interativo:** Oferece um menu de opções para navegação fácil entre as funcionalidades.
- **Validação de Entrada:** Verifica se as entradas do usuário são válidas e exibe mensagens apropriadas para opções inválidas.

## Como Usar

1. Clone este repositório:
    ```sh
    git clone https://github.com/seu-usuario/banco-ficticio-aaa.git
    ```

2. Navegue até o diretório do projeto:
    ```sh
    cd banco-ficticio-aaa
    ```

3. Compile e execute o programa:
    ```sh
    javac Main.java
    java Main
    ```
